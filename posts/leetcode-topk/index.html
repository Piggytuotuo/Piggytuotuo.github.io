排序-TopK

Posted Apr 26, 2023
By Crystal Li
2 min read

使用QuickSort和HeapSort解决TopK问题。

QuickSort O(NlogN)

定义快排的partition操作：使用a[l]作为切分元素，从左向右扫描数组，找到第一个>= a[l]的元素，再从右向左扫描数组，找到第一个< a[l]的元素，交换这两个元素，当两个指针相遇，将切分元素a[l]和a[j]交换位置。

public static int partition(int[] nums, int l, int h) {
	//扫描数组，找到第一个>=al的 和 第一个< al的
	int i = l, j = h + 1;
	int flag = nums[l];
	while(true) {
		while(nums[++i] < flag && i < h);
		while(nums[--j] >= flag && j > l);
		if(i >= j) break;
		swap(nums, i, j);
	}
	swap(nums, l , j);
	return j;
}

public static void swap(int[] nums, int i, int j) {
	int a = nums[i];
	nums[i] = nums[j];
	nums[j] = a;
}

根据partition操作，递归写快排。

public static void quickSort(int[] nums, int l, int h) {
	if(l >= h) return;
	int j = partition(nums, l, h);
	quickSort(nums, l, j -1);
	quickSort(nums, j + 1, h);
}

QuickSort找Top-K O(N)

利用partition操作，根据返回的j的位置判断K元素在的位置，缩小下次partition的边界。

在升序序列中，第K大元素的坐标为：N-K。

public static int topK(int[] nums, int k) {
    k = nums.length - k; 
	int l = 0, h = nums.length - 1;
	while(l < h) {
		int j = partition(nums, l, h);
		if(j == k) {
			break;
		}else if(j < k) {
			l = j + 1;
		}else {
			h = j - 1;
		}
	}
	return nums[k];	
}

HeapSort O(NlogK)

找第K大元素，维护一个大小为K的小顶堆，堆顶即为所求。

public static int heapTopK(int[] nums, int k) {
	Queue<Integer> pq = new PriorityQueue<>();
	for(int i: nums) {
		pq.add(i);
        //维护堆的大小为k
		if(pq.size() > k) {
			pq.poll();
		}
	}
	return pq.peek();
}
